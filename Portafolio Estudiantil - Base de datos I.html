Portafolio Estudiantil – Base de Datos I

Este sitio reúne las actividades, evaluaciones, materiales y evidencias realizadas durante el curso Base de Datos I, con énfasis en el diseño y manejo de bases de datos relacionales. El propósito es documentar el proceso de aprendizaje del grupo, desde las tareas iniciales hasta el proyecto final.

Acerca del sitio, créditos y tecnologías utilizadas

Cómo se construyó el portafolio y herramientas usadas durante el semestre.

Acerca del sitio

Este portafolio fue elaborado como requisito de evaluación del curso Base de Datos I. Su objetivo es organizar en un solo lugar actividades, materiales y evidencias del semestre.

Créditos

El contenido académico fue desarrollado por los estudiantes del grupo 2IL-121. La estructura del sitio fue diseñada para facilitar la revisión y descarga de evidencias.

Tecnologías utilizadas

  • HTML5 para estructura del contenido.
  • CSS3 para diseño y organización visual.
  • GitHub Pages para publicar la página.
  • OneDrive/SharePoint para alojar PDFs, imágenes y videos.
  • Microsoft SQL Server como herramienta de práctica.
